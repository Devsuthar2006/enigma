Process

How it works

  1. Create a room

    Pick a topic, choose a discussion mode, and generate a unique room code.

  2. Share the code

    Participants scan the QR code or enter the room code to join instantly on any device.

  3. Record arguments

    Each participant records their audio argument during their turn. AI transcribes in real-time.

  4. AI evaluates

    Every argument is scored on logic, clarity, relevance, and emotional bias using GPT-4.

  5. Review results

    View scores, analytics, insights, and download a full session report.
